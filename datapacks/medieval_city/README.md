# Medieval City datapack — Enhanced variants added

This branch expands the modular medieval generator with many more building and decoration variants:

New building variants
- gothic_house: tall vertical house with pointed roof and deepslate accents
- mudbrick_house: warm mud-brick / terracotta-style house and thatch accents
- deepslate_roof_house: heavy deepslate tiled roof variant
- blacksmith_improved: larger forge with chimney, smoker, and horse stable
- tavern_improved: larger tavern with signage, multiple guest rooms and kitchen
- castle_gothic: gothic castle variant with pointed towers, iron bars, and chains
- market_stall_variant2: alternate stall layout and awnings

New decorations & props
- lantern_post: street lamp using chains + lanterns
- well: decorative well with bucket
- bench: seating using slabs and trapdoors
- large_tree: small custom tree for village squares

How to use
- /function medieval:generate/random_building — will now include the new variants
- /function medieval:generate/city_center — will produce a richer layout if you run it again
- /function medieval:setup_scoreboards — run after reload if you haven't already

If you want further expansion (more roofs, interior furnishings, or conversion to .nbt structures), tell me which targets and I will continue.
